## RESTful Routes

|   NAME   |     PATH              |    VERB         |            DESCRIPTION               |
|----------|-----------------------|-----------------|--------------------------------------| 
| Index    | /campgrounds          |      GET        | Displays all campgrounds             |
| New      | /campgrounds/new      |      GET        | Shows form for new campground entry  |
| Create   | /campgrounds          |      POST       | Creates a new campgrounds            |
| Show     | /campgrounds/:id      |      GET        | Shows one specified campground       |
| Edit     | /campgrounds/:id/edit |      GET        | Shows edit form for one campground   |
| Update   | /campgrounds/:id      |      PUT        | Updates a particular campground      |
| Destroy  | /campgrounds/:id      |      DELETE     | Deletes a particular campground      |
